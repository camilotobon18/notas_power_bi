POWER BI
empezo con el complemento power pivot, query, view (graficos interactivos|) y map en Excel 2013.
Nace en 2015
combina la tecnología de el software de administración de datos mas usado del mundo (SQL Server) con el software de BI más popular del planeta (Excel) y mucho más


BI

INFORME
coleccion de visualizaciones que se muestran en una o varias paginas y q normalmente hacen referencia a elemetos y datos relacionados entre si.

PANELES
colecciones de objetos visuales q se importan de uno o mas informes que ya existen.
ocupan una unica pagina, normalmente representa informacion de alto nivel

ELEMENTOS
POWER BI DESKTOP
	gratuita, se descarga e instala. no permite creacion de paneles(unicamente en el service)
POWER BI SERVICE
	SaaS, ofrecido en la nube, compartición de informes, analisis automatico de los datos. tiene costo.
POWER BI MOBILE
	principalmente se utiliza para visualizar

FLUJO DE TRABAJO
	DESKTOP
		Cargar conjunto de datos
		transformarlos
		crear un informe
		publicar
	SERVICES
		Crear panel
		compartirlo a los usuarios
	MOBILE
		Acceder al informe

LENGUAJE DAX
	lenguaje de modelado de datos
	-columnas calculadas (estaticas) se mantienen estaticas a menos q se modifique la formula
	-medidas (dinamicas) se calculan cada vez que se utilizan
