-ADMINISTRAR
  Eliminar, la consulta o tabla en la que estamos parados

-ELEGIR COLUMNAS
  Elegir columnas, permite seleccionar las columnas que se desean conservar
  Ir a columna, permite seleccionar la columna en la que nos queremos parar, sobretodo para tablas muy grandes

-QUITAR COLUMNAS 
  Quitar columnas, quita las que seleccionamos
  Quitar otras columnas, quita las que no se seleccionaron
  
-CONSERVAR FILAS
podemos elegir entre:
  conservar filas superiores (se define el número)
  conservar filas inferiores (se define el número)
  conservar intervalo de filas (1ra fila,
                                número de filas)
  conservar duplicados
  conservar filas en blanco (null)
  conservar errores
  
-QUITAR FILAS
podemos elegir entre:
  quitar filas superiores (se define el número)
  quitar filas inferiores (se define el número)
  quitar filas alternas (1ra fila q se desea quitar,
                         numero de filas que se desean quitar,
                         numero de filas que se desean conservar)
  quitar duplicados
  quitar filas en blanco (null)
  quitar errores
